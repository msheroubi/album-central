# Album Central
Just a fun project to compare album ratings between a group of friends.

#### Requirements
``` 
django
django-tinymce
django-bootstrap-modal-forms
selenium
```

### Functionality
* Browse albums by genre
* Upload new album using album and artist name
* Cover art, track list and lyrics are all automatically scraped from the web
* Rate albums and individual tracks
* Account portal to see and modify personal ratings
* Calculate overall rating based on user ratings
